The Three Fires

One install.
Three powerful tools.

Prometheus ships as a CLI, a GitHub Action, and a VS Code extension — all governed by the same 505-rule engine.

⌨️
CLI — The Oracle

prometheus scan deep-scans your repo. prometheus review evaluates staged changes. Reports health grades, severity breakdowns, and actionable findings — in terminal or JSON.

⚙️
GitHub Action — The Guardian

Drop one YAML file into your repo. Every PR gets a governance review automatically. BLOCKER findings block the merge. Inline comments pinpoint the exact line. No tokens. No secrets. No setup beyond copy-paste.

🔌
VS Code Extension — The Watcher

Real-time findings in your editor as you write. Health score in the status bar. The AI adapters panel syncs your governance context to Claude, Gemini, Cursor, Copilot, and Codex — so they write governed code from the start.

📦
Pack Ecosystem — Infinite Fire

The 505 built-in rules are only the beginning. Install community packs or author your own. Drop a pack into .prometheus/packs/ and it's live on the next scan. No rebuild. No config change.

📊
Health Scoring — Know Your Grade

Every scan produces a health grade: A+, A, B, C, D, or F. Track it over time. Gate deploys on it. Post it in every PR. The whole team sees the same signal — no more subjective code review debates.

⚡
