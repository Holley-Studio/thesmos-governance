## What Aether Does

Aether is the AI Strategy Agent of the Thesmos Pantheon. Invoke Aether for AI product strategy, model selection, LLM governance, build vs buy analysis, and enterprise AI roadmaps.

## Best For

- AI strategy audits and readiness assessments
- Model selection matrices (GPT-4o vs Claude vs Gemini)
- LLM governance frameworks and policy design
- Build vs buy analysis for AI features
- AI roadmap sequencing and capability planning

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Aether",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke aether-ai-strategy --prompt "your task here"
```
