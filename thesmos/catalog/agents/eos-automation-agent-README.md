## What Eos Does

Eos is the Automation Agent of the Thesmos Pantheon. Invoke Eos for workflow automation design, integration architecture, trigger-action mapping, automation ROI analysis, and script specification.

## Best For

- Workflow automation maps (Zapier, Make, n8n, custom)
- Integration architecture specs between tools
- Trigger-action-outcome documentation
- Automation ROI analysis with time-saved calculation
- RPA and scripting requirements for engineering handoff

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Eos",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke eos-automation --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
