# Zeus Orchestration Guide

God Agent Zeus is the executive orchestrator of the Thesmos Pantheon. He does not do the work — he routes it to the right specialist, ensures every initiative has a clear owner, and aligns output to your business's North Star.

This guide tells you exactly which agent to invoke for every type of task.

---

## The Core Principle

**Zeus first, then the specialist.** When you're unsure which agent to use, start with Zeus. He will assess your situation, identify the right specialist, and hand off with context so the specialist can hit the ground running.

```
"God Agent Zeus, I need to [describe your goal]. Who should I work with and what should I tell them?"
```

## Chaining Agents

The real power of the Pantheon is chaining agents for complex initiatives. Zeus coordinates multi-agent chains.

### Example: Product Launch

```
Zeus → Athena (strategy) → Daedalus (PRD) → Talos (engineering brief) → Apollo (launch content) → Hermes (campaign) → Ares (sales enablement)
```

**Invocation pattern:**

```
"God Agent Zeus, I'm launching [product] to [audience] in [timeframe]. 
Assess the situation and tell me which agents to invoke in what order."
```

### Example: New Business Vertical

```
Zeus → Athena (positioning) → Nike (ICP + outbound) → Ares (pitch) → Plutus (unit economics) → Hera (operations)
```

### Example: Brand Relaunch

```
Zeus → Aphrodite (visual identity) → Erato (brand voice) → Apollo (messaging) → Hermes (go-to-market) → Pheme (PR)
```

## Momus: The Most Underused Agent

Before any major plan executes, invoke Momus. He challenges assumptions, exposes blind spots, and asks the questions the team didn't think to ask.

```
"God Agent Momus, challenge this plan before we execute it: [paste plan]"
```

Zeus will often route to Momus proactively. But you can invoke him directly whenever you sense a plan hasn't been pressure-tested.

---

## Proteus: Drift Detection

Proteus monitors when you've strayed from the original plan — product scope creep, prompt drift, brand inconsistency, strategy drift.

Invoke him at major milestones:
```
"God Agent Proteus, compare what we're building now to the original [product spec / brand brief / strategy]. What has drifted and should we course-correct?"
```

## The God Council

For high-stakes decisions with multiple competing priorities, convene the God Council:

```
"God Agent Zeus, convene the God Council for [decision]. 
I need Athena's strategy assessment, Momus's challenge, and Metis's execution plan."
```

Zeus will sequence the session, compile the outputs, and deliver a unified brief.

## Tips for Maximum Output Quality

1. **Context is everything.** The more you tell an agent about your company, audience, and constraints, the sharper the output.
2. **One task per agent.** Don't ask Ares to write a sales pitch AND a marketing strategy. Route to Ares for sales, Hermes for marketing.
3. **Trust the judgment, not just the output.** Each agent's `What makes my judgment different` section explains the non-obvious insights they apply. Read it once per agent.
4. **Thesmos governs everything.** If you're using Claude Code, every agent output is checked against 1,075 governance rules before it reaches you.
5. **Zeus doesn't do the work.** If Zeus gives you content, he's cheating. He should route — not produce.
