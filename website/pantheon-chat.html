What it is

Governance and a god council, in the editor

Thesmos is an AI governance and multi-agent orchestration platform built for teams shipping code with Claude, Cursor, Copilot, or Gemini. Getting started

Three steps

Install the extension. Search "Thesmos Governance" in the VS Code Extensions panel, or install from the Marketplace page.
Add your Anthropic API key. Open Pantheon Chat (Cmd/Ctrl+Shift+G) and enter your key when prompted. It's stored in VS Code secret storage — never in plaintext.
Run your first scan. From the Command Palette, run Thesmos: Scan Repository. Findings appear in the sidebar tree — right-click any one to fix it inline or copy an AI fix prompt.

Requirements

VS Code 1.100.0 or later · an Anthropic API key for chat and AI fixes · Node.js in PATH for CLI scan commands (optional — UI scan works without it) · a .thesmos/config.json in your project root, generated on first scan.
