Setup

Governed in minutes.
Protected forever.

No account. No cloud. Prometheus runs locally, in CI, and in your editor — simultaneously.

I

Install

Run npm install prometheus-governance. All 505 rules activate immediately — no config file needed.

II

Scan

Run prometheus scan to get your first health grade and see every finding in your codebase, categorized by severity.

III

Protect CI

Copy one YAML file to .github/workflows/. Every PR from this point is governed — BLOCKERs block the merge automatically.

IV

Extend

Write custom rules or drop community packs into .prometheus/packs/. They load at runtime, no rebuild needed.
