# Pantheon Pro Kit

Welcome to Pantheon Pro. You have all 34 God Agents ready to deploy.

## Quick Start

1. Choose your platform from the `setup/` folder
2. Follow the guide for your AI tool (Claude Code, ChatGPT, Cursor, etc.)
3. Read `setup/zeus-orchestration-guide.md` to learn which agent to invoke when
4. Invoke your first agent: *"God Agent Zeus, I need to prioritize my roadmap for Q3."*

## What's In This Kit

| Folder | Platform | Format |
|---|---|---|
| `claude-code/` | Claude Code | `.md` (copy to `.claude/agents/`) |
| `claude-project/` | Claude.ai Projects | `.txt` (paste as Project Instructions) |
| `chatgpt/` | ChatGPT Custom GPTs | `.txt` (paste as system prompt) |
| `cursor/` | Cursor | `.mdc` (copy to `.cursor/rules/`) |
| `gemini/` | Gemini | `.txt` (paste as system instructions) |
| `copilot/` | GitHub Copilot | `.md` (use with Copilot instructions) |
| `openai-assistants/` | OpenAI Assistants API | `.json` (use `instructions` field) |
