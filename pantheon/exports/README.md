# Pantheon Exports

This directory contains the **free Starter Council** — 5 fully-functional god
agents (⚡ Zeus, 🦉 Athena, 👁 Argus, ✍️ Apollo, 🔨 Hephaestus) exported for
ChatGPT, Claude Code, Cursor, and Gemini — plus the source for the free
[Pantheon Council GPT](https://holley.studio/go/gpt.html). No account, no
email, no payment. Grab the files, drop them into your tool, done.

## Where are the other 62 agents?

The full Pantheon (67 specialist agents × 10 platforms, Zeus orchestrators,
ChatGPT knowledge clusters, Claude council bundles, one-click installers, and
lifetime updates on every god drop) is the paid pack:

**→ [holley.studio/thesmos](https://holley.studio/thesmos)** — one-time
purchase, no subscription.

We keep the split honest in both directions: the CLI, all 1,137 governance
rules, and these 5 starters are free and public, forever. The packaged
Pantheon is how this project gets funded. Older revisions of this repository
contained more of the export tree — the license (FSL-1.1-MIT, see
[LICENSE](../../LICENSE)) does not permit redistributing or reselling that
content.
